# 截图说明

此目录用于存放项目效果展示截图。

## 截图列表

请将以下截图放在此目录下：

1. **home.png** - 主页截图
   - 展示三个功能卡片并排布局
   - 突出产品定位和视觉设计

2. **chat.png** - 智能对话截图
   - 展示完整的对话流程
   - 包含普通聊天、历史查询、工具调用等场景

3. **diary.png** - 成长日记截图
   - 展示日记记录界面
   - 包含心情选择、标签管理、内容输入

4. **task-assistant.png** - 任务助手截图
   - 展示智能体执行复杂任务的流程
   - 展示多步骤推理和执行过程

5. **features.png** (可选) - 功能演示截图
   - 展示智能路由提示
   - 展示 RAG 检索结果
   - 展示工具调用过程

6. **responsive.png** (可选) - 响应式设计截图
   - 展示不同屏幕尺寸下的布局

## 截图要求

- **分辨率**：建议 1920x1080 或更高
- **格式**：PNG 或 JPG
- **命名**：使用英文命名
- **质量**：清晰、美观，突出产品特色
